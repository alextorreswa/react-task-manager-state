# Task Manager with React State

This project is a React application built with Vite that demonstrates how to manage arrays and objects in React state by creating a simple task manager.

## Features

- Add new tasks
- Display task title and completion status
- Toggle tasks between pending and completed
- Prevent empty tasks from being added
- Simple responsive styling

## Concepts Demonstrated

- React `useState`
- Arrays in state
- Objects in state
- Immutable updates with spread syntax
- Updating items with `map()`
- Conditional rendering

## How to Run

1. Clone the repository
2. Open the project folder in VS Code
3. Install dependencies
4. Start the development server
5. Open the local development link in your browser 

## Project Structure
- src/TaskManager.jsx - Main task manager component
- src/TaskManager.css - Styling for the app
- src/App.jsx - Renders the TaskManager component
